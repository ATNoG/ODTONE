Changelog

ODTONE 0.6 - Open Dot Twenty ONE

  • Fix transaction ID serialization/deserialization and improve its robustness
  • Improve robustness of IPv4/IPv6 encoding and decoding
  • Include a JAVA MIH Protocol library
  • Move some extensions to submodules (check documentation)
  • Merge all ODTONE libraries ('base', 'mih', 'sap' and 'net') in a single library (libodtone)
  • Add MIH Resource Adaptor for Mobicents JAIN SLEE (https://github.com/ATNoG/MIH-Resource-Adaptor)
  • Add OpenWrt ODTONE package (https://github.com/ATNoG/odtone-openwrt)
  • ODTONE website and documentation revision.
