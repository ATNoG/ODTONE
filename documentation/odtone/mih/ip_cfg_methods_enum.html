enum ip_cfg_methods_enum { ip_cfg_ipv4_static =  0, ip_cfg_ipv4_dynamic =  1, 
                           ip_cfg_ipv4_mobile_fa =  2, 
                           ip_cfg_ipv4_mobile =  3, 
                           ip_cfg_ipv6_stateless =  11, 
                           ip_cfg_ipv6_stateful =  12, 
                           ip_cfg_ipv6_manual =  13 };

Description

IP_CFG_METHODS data type enumeration.

ip_cfg_ipv4_static

IPv4 static configuration.

ip_cfg_ipv4_dynamic

IPv4 dynamic configuration (DHCPv4).

ip_cfg_ipv4_mobile_fa

Mobile IPv4 with FA-CoA.

ip_cfg_ipv4_mobile

Mobile IPv4 without FA (Co-located CoA).

ip_cfg_ipv6_stateless

IPv6 stateless address configuration.

ip_cfg_ipv6_stateful

IPv6 stateful address configuration (DHCPv6).

ip_cfg_ipv6_manual

IPv6 manual configuration.
