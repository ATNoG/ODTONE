Type dns_query_type

odtone::dns::dns_query_type

Synopsis

// In header: </home/carlos/Projectos/odtone/inc/odtone/net/dns/resolver.hpp>



enum dns_query_type { DNS_A_RECORD =  0x01, DNS_MX_RECORD =  0x0f, 
                      DNS_AAAA_RECORD =  0x1c, DNS_SRV_RECORD =  0x21, 
                      DNS_NAPTR_RECORD =  0x23 };

Description

DNS query type enumeration.

DNS_A_RECORD

Lookup IPv4 address for host.

DNS_MX_RECORD

Lookup MX for domain.

DNS_AAAA_RECORD

Lookup IPv6 address for host.

DNS_SRV_RECORD

Lookup SRV for domain.

DNS_NAPTR_RECORD

Lookup NAPTR for domain.
