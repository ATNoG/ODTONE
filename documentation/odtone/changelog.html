Changelog

ODTONE 0.5 - Open Dot Twenty ONE.

  • Provision of extended support Linux and Windows IEEE 802.11 Link SAPs.
  • Provision of extended support Linux Ethernet Link SAP.
  • Provision of a C# MIH Protocol library.
  • Update of the DHCP MIH-User to the last version of ISC's DHCP (4.2.5).
  • Extended support for the MIH Registration signaling added.
  • Added CMake build support.
  • Provision of *.deb packages.
  • Several bug-fixes.
  • ODTONE documentation revision.
